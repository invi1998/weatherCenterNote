# 阻塞和非阻塞IO

- 阻塞：程序在调用结果返回之前，会等待，得到结果之后才会返回
- 非阻塞：不管调用是否得到结果，程序都不会等待，立即返回
- 缺省阻塞的函数：`connect()`,`send()`,`recv()`,`accept()`

```c++
 #include <unistd.h>
 #include <fcntl.h>

 int fcntl(int fd, int cmd, ... /* arg */ );

```

使用 `fcntl()`这个函数可以设置socket文件描述符为非阻塞。

```c++
fcntl(sockfd, F_SETFL, O_NONBLOCK);		// 把socket连接设置为非阻塞
```
